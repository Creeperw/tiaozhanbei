# 智能助教侧栏与输入区收敛设计

**目标：** 缩小历史记录侧栏和“新对话”按钮，并将底部输入区从多层框线收敛为一个外层圆角输入容器。

## 设计结论

- 会话侧栏展开宽度由 280px 调整为 244px，列表文字继续使用截断，编辑与删除入口保持不变。
- “新对话”按钮高度调整为 36px，缩小横向内边距、圆角和图标，仍保留清晰点击反馈。
- 移除消息区与输入区之间的顶部分隔线，让输入区自然贴合页面背景。
- textarea 不再显示独立边框或 `focus-visible` 外框；焦点反馈统一落在最外层圆角容器上。
- 保留文件拖放、工具菜单、语音、发送及键盘提交行为。

## 验证

- 组件测试确认侧栏宽度、按钮高度及 textarea 的无内框类名。
- 运行智能助教组件测试、ESLint 和生产构建。
