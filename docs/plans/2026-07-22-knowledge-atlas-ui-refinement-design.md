# 知识仓库与助教界面收敛设计

**目标：** 删除首页和助教页冗余操作，收紧助教标题栏，并让知识星球默认顺序浏览、球面视觉更轻巧、三级节点与聚类布局更自然。

## 设计结论

- 首页移除 Hero 右上角“上传资料”和“开始学习”，保留主体内容与功能卡片导航。
- 助教顶部仅移除“新对话”，侧栏关闭时仍显示“展开侧边栏”；标题栏高度控制在 40–44px，标题保持水平居中。
- 知识星球初始布局改为“顺序”，球面和聚类仍可手动切换。
- 球面节点半径整体降低约 20%，标签字号降到 10–11px，取消标签白色矩形底，只增加轻微文字阴影以保证可读性。
- 三级节点统一使用实心基底，通过金色环、中心点等符号区分题目和视频资源，不再出现空心或虚线球。
- 聚类采用可见正面的均衡簇中心，限制纵向分布并降低聚类预设缩放，避免节点切换后沉到画布底部。

## 验证

- 组件测试确认首页按钮和助教“新对话”不再渲染、知识星球默认顺序视图。
- 模型和画布测试确认聚类坐标在安全范围、三级节点均具备实心基底。
- 运行相关 Vitest 测试与 Vite 构建。
