# 前端产品化适配设计

## 目标

在不改变现有数据流测试页、不迁移 React/Tailwind 技术栈的前提下，把正式前端从功能演示状态升级为可连续使用的学习产品。后端稳定契约继续作为数据来源，只有在前端无法正确完成入口访问或缺少必要展示字段时才做最小后端调整。

## 产品边界

- 正式多智能体口径固定为六个角色：任务规划、记忆管理、学情诊断、知识库管理、专家、审核裁判。
- 默认路线解析、学习计划落库、复习调度、试卷蓝图与组卷、工具调用均为智能体内部阶段或确定性服务，不显示为第七个智能体。
- 前端优先展示自然语言。系统 ID、工具参数和原始事件仅在技术详情中按需展开。
- 保留现有用户隔离、认证、页面意图与后端接口，不新增前端业务真相。
- `/demo/` 数据流测试界面不修改。

## 界面设计

### 对话工作区

对话页采用会话列表、主对话、执行详情三层布局。会话列表可折叠；主对话区承担问题、自然语言回答、引用、资源和业务跳转；执行详情默认为紧凑状态，不永久占据大块屏幕。

用户消息保持轻量，智能体回答使用正文排版而不是大面积聊天气泡。输入框固定在主对话区底部，滚动只发生在消息区。上传、停止生成、重试、复制、反馈和业务跳转均提供明确标签与禁用状态。

### 六智能体任务台

六个智能体始终保留稳定席位。本次参与者按实际事件顺序显示等待、执行、完成、失败、等待追问、已跳过状态；未参与者显示“本次无需参与”。

对用户的阶段名称固定为：

1. 任务规划：理解需求并安排执行路径。
2. 记忆管理：读取会话、偏好和有效历史。
3. 学情诊断：判断掌握状态和学习节奏。
4. 知识库管理：检索教材、题目、视频和证据。
5. 专家：生成计划、讲解、题目或试卷。
6. 审核裁判：检查事实、质量与发布条件。

技术名称、耗时、工具调用、参数、重试和原始事件放进每个智能体的展开详情。空参数调用和重复事件不进入用户摘要。

### 训练工坊

训练工坊保留题目训练、知识卡片、试卷生成三个正式模块。模块导航表达“现在能做什么”，内容区表达“当前正在做什么”，训练产物不再作为无差别空卡片常驻。

- 题目训练统一承载题目作答与批改、案例训练、错题变式。
- 知识卡采用资料目录与阅读详情布局，按讲解、教材切片、视频、题目组织内容，并标明来源与补充检索状态。
- 试卷生成采用试卷列表与答题工作区，突出总分、题量、限时、保存状态、交卷状态和剩余时间。

### 滚动与响应式

应用外壳只负责视口布局，各业务页面明确自己的滚动容器。桌面端训练工坊、知识卡、试卷和长对话必须能够完整纵向滚动；侧栏与输入区保持独立。移动端取消多列强制高度，按自然文档流排列。

不得出现可视区内容高度大于容器但 `overflow-y: hidden` 的板块。弹层、详情抽屉和技术轨迹各自拥有受控滚动区域。

## 状态与错误处理

- 未登录访问 `/` 时必须能够加载 React 登录页；静态 JS、CSS、字体、图像和 favicon 不得被认证边界拦截。
- 首次用户没有画像、计划或训练记录时显示下一步行动，不显示伪造进度。
- 接口加载使用骨架或明确的加载文案；失败状态保留重试入口和可读原因。
- 智能体追问、运行中、停止、失败、审核拒绝和成功发布使用不同状态，不用同一个通用提示替代。
- 页面跳转继续使用现有 `pageIntent`；从智能体产物进入知识卡或试卷时保留目标 ID。

## 数据流

运行事件先经过纯函数映射：内部 agent key 转成六智能体角色，服务事件归入所属角色，重复事件合并，用户摘要与技术详情分离。视图组件只消费映射后的展示模型，不在 JSX 内推断业务归属。

知识卡、试卷和训练模块继续消费现有稳定接口。后端仅补充静态资源公开白名单及前端确实需要但契约缺失的字段。

## 可访问性与细节

- 所有图标按钮提供可访问名称，弹层支持焦点进入、Escape 关闭和焦点归还。
- 状态不只依赖颜色，计时和数字使用等宽数字。
- 交互控件满足清晰的 hover、focus、active、disabled 状态。
- 文案统一使用中文用户语言，不在主界面暴露 `Planner`、`Executor`、`Tool Calls`、`running` 等内部词汇。
- 动效尊重 `prefers-reduced-motion`。

## 测试与验收

- 先为静态资源公开访问、六智能体映射、重复事件折叠、桌面滚动、空状态与跨模块跳转编写失败测试。
- 前端单元/组件测试、生产构建与 lint 必须通过。
- 后端正式离线测试必须通过；Live pytest 不从 WSL 命令行运行。
- Live 验收通过已启动前端的 Execute 和 Playwright 页面交互完成，覆盖登录入口、正式对话、六智能体任务台、训练工坊滚动、知识卡与试卷跳转。
