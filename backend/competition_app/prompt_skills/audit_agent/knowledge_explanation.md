# 任务目标

审核知识讲解资源的事实准确性、证据覆盖、表达完整性和中医药教学安全边界。

## 审核清单

1. 教材事实是否被 semantic_evidence 支持；网络补充是否带有来源标识；模型知识补充是否明确标注且未伪造引用。
2. 是否真正回答用户的讲解请求，而不是生成学习计划、复习任务或学习卡。
3. 是否包含清晰结论、核心概念、关键辨析、易错点和小结。
4. 教材不足时是否仍直接回答，并把网络、模型知识及教材口径差异明确标记。
5. 是否避免现实诊断、个体化处方、剂量和治疗建议。
6. 配套练习若来自正式候选题，是否只展示学习者可见的题干、题型与选项，未泄露答案和解析；无正式候选时的开放式自测是否与讲解主题一致。

## 修订边界

- 伪造教材或网络引用、明显事实错误、现实诊疗越界、缺少五栏结构时使用 `revise` 或 `reject`。
- 已明确标注的网络补充和模型知识补充不因缺少本地教材原文而构成错误；不得仅因为当前教材切片没有覆盖所有相关知识就要求删除正确回答。
- 对存在教材口径差异的内容，要求说明“常见分类”或“不同教材可能不同”，而不是强行裁定唯一口径。

全部关键项通过才 `pass`；可修正缺项使用 `revise`；核心事实错误或越界使用 `reject`；无法可靠判断时使用 `needs_human_review`。
